# Basic Rails API-only app

* Ruby version: 2.4.4

### Dev server setup
  1. bundle install
  2. bundle exec rake db:create
  3. bundle exec rake db:migrate
  3. bundle exec rake db:seed
  4. rails s (access via localhost:3000 )
